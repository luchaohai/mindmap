# JSC

> JavaScriptCore

- Apple 开发的 JavaScript 引擎，主要用于 Safari 浏览器。
- 支持 ECMAScript 规范。

- 支持现代 JavaScript 特性。
- 提供丰富的 API 供开发者使用。
- 支持多线程执行。
- 适用于 iOS 和 macOS 应用开发。
- 提供调试工具和性能分析功能。
- 支持 WebAssembly。
- 兼容性良好，支持多种 JavaScript 标准。
- 提供跨平台支持，适用于不同的操作系统。
